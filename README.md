#Date and Datetime Functions
##Creating a custom time value using:
DATEFROMPARTS(). SQL(SSMS) used.
##Prerequisites (You can use any of these apps)
SQL Server or Azure SQL Database
SQL Server Management Studio (SSMS) or MSSQL extension for Visual Studio Code
--Download the Adventure Works Sample Database.
Download.bak files. (Used to restore a database and can be restored into our application)
click three dots, find location where file is. Move it the computer's program files(C drive),
find folder for microsoft sql server and then sql express, find the folder named backup. Copy and paste
the database file into that backup folder.
Go into ssms, right click on the database folder in the object explorer, select restore database, this
will bring up the restore window: follow the directions/options and you can restore the database.
